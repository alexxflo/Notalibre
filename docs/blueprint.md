# **App Name**: SalvaFans

## Core Features:

- Initial Gatekeeper Modal: Display a full-screen modal for new users, prompting them to follow @AlexxIA on TikTok. The modal includes a 20-second countdown before the 'Confirm' button is enabled, after the TikTok follow url opens. The TikTok follow url includes the strings 'is_from_webapp' and 'sender_device'.
- Coin Balance: Display the user's current coin balance.
- Follow Exchange: Enable users to give and receive follows, subtracting 5 coins for receiving one follower and adding 3 coins for giving one.
- Welcome Bonus: Grant new users a welcome bonus of 50 coins.
- Coin Purchase Options: Display available coin packages: 20 coins ($20 MXN), 50 coins ($35 MXN), 100 coins ($50 MXN).
- Payment Instructions: Show the bank account number '638180000106470075' for coin purchases.

## Style Guidelines:

- Primary color: Saturated blue (#29ABE2), reflecting trustworthiness and connectivity in a traffic exchange platform.
- Background color: Light blue (#E5F5FF), offering a clean and airy backdrop.
- Accent color: Vivid cyan (#00FFFF), providing prominent contrast to the analogous blue palette.
- Font pairing: 'Poppins' (sans-serif) for headlines and short amounts of text, and 'PT Sans' (sans-serif) for body text. Note: currently only Google Fonts are supported.
- Use clear and concise icons to represent coin transactions and user interactions.
- The Gatekeeper modal should cover the entire screen and be the very first experience in the app.
- Subtle transitions and animations on the main pages to enhance user experience without being distracting.
